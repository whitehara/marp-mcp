## What's Changed

### Bug Fixes
- Fix aspect ratio distortion in `image-center` and `figure-caption` layouts when image width exceeds 1200px (was incorrectly specifying both `h:` and `w:` constraints simultaneously)

### Improvements
- Move frequently-adjusted font-size values from CSS to inline styles in generated slide HTML for easier per-slide editing
  - `methodology`: step label (0.9em) and step description (0.75em)
  - `statistics`: stat number (2.2em) and stat label (0.8em)
  - `results-table`: table font size (0.85em)
  - `sidebar`: sidebar panel font size (0.8em)
